## 走査電子顕微鏡（SEM）のデータ構造化
サンプルデータは卓上電子顕微鏡（低真空SEM）で複数の倍率で撮影されたファイルです．一連の撮影情報を測定メタデータのtxtファイルからをリスト化します．

**目　標**：　ここでは，データ構造化の処理の一つである**測定メタデータの抽出**の基本的な処理の流れを習得することを目指します．

**機　種**：　日立ハイテク　TM3000, TM4000シリーズ   
**データ**：　txt (テキスト形式)
