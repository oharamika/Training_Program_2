## 走査電子顕微鏡（SEM）のデータ構造化
サンプルデータは卓上電子顕微鏡（低真空SEM）で複数の倍率で撮影されたファイルです．撮影情報をリスト化するデータ構造化のスクリプトに関する動作方法について説明します．

ここでは，データ構造化の処理の一つである「メタデータの抽出」の基本的な処理の流れを習得することを目指します．

**対応機種** ：　 日立ハイテク　TM3000, TM4000シリーズ   
**rawデータ**：　txt (テキスト形式)
